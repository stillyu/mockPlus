# Mock Plus - web测试的利器
Mock Plus是一个Chrome扩展，主要用于生成虚假身份证号码或其他个人信息，用于web测试。

Mock Plus具有两种使用方式：**Chrome右上角的扩展按钮**和**页面的右键菜单**。
